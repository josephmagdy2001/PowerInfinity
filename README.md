# Power Infinity

A modern and responsive corporate landing page built with **React.js** to showcase the services, expertise, and vision of **Power Infinity**, a software development company. The website is designed to provide potential clients with a professional overview of the company's solutions while delivering a fast and seamless browsing experience.

---

## 🌐 Live Demo

**Demo:** https://powerinfinitywebsite.vercel.app/

---

# 📖 Overview

Power Infinity is a responsive company website designed to present software development services, highlight technical expertise, and strengthen the company's online presence.

The project focuses on clean UI, responsive layouts, reusable React components, and modern frontend development practices.

# 🚀 Getting Started

Clone the repository.

```bash
git clone https://github.com/josephmagdy2001/power-infinity.git
```

Navigate to the project directory.

```bash
cd power-infinity
```

Install dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```

Build for production.

```bash
npm run build
```

Preview the production build.

```bash
npm run preview
```
